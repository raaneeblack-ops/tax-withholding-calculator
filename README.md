# Restaurant Information Application

## Overview
This is a simple Java application that demonstrates basic programming concepts by displaying restaurant information.

## Files
- `RestaurantInfo.java` - Main Java source code
- `pseudocode.txt` - Pseudocode algorithm
- `Restaurant_Info_Assignment.docx` - Complete assignment documentation

## Restaurant Information Displayed
- Restaurant Name: The Olive Garden
- Business Address: 1234 Restaurant Blvd
- City, State, Zip: Lancaster, California 93534

## How to Compile
```bash
javac RestaurantInfo.java
```

## How to Run
```bash
java RestaurantInfo
```

## Expected Output
```
Restaurant Information:
======================
Restaurant Name: The Olive Garden
Business Address: 1234 Restaurant Blvd
City, State, Zip: Lancaster, California 93534
```

## Programming Concepts Demonstrated
- Variable declaration and initialization
- String data types
- String concatenation
- Console output (System.out.println)
- Code comments and documentation
- Java main method structure
